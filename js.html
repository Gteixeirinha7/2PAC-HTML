Javascript e suas "Arte manhas"

Aprimore seus conhecimentos e impressione outros desenvolvedores, Março 10, 2020

JavaScript é uma das linguagens mais famosas da WEB, e muito conhecida no mundo da tecnologia, mas porque?

Vamos citar 2 "manhas" que so o Javascript permite voce a fazer

  • Chamada de metodo em operador ternario.

    Praticamente 100% dos programadores utilizando ou sabem oq é um if ternario, aquela instrução de decisão que pode ser executada em uma linha:

    (VOCÊ CONHECE JAVASCRIPT ? SIM : NÃO)

    Agora imagine fazer isso e chamar um metodo, podemos fazer desta forma:


    funtion x(){
    console.log('Olá Mundo X');
    }

    funtion y(){
    console.log('Olá Mundo Y');
    }

    //Chamando a função

    (true ? x : y)();

    Com isso, podemos chamar qualquer tipo e nome de metodo desta forma, so cuidado para não escrever errado o nome ein....

  • Template Literals

    Como voce faz hoje para colocar uma variavel dentro de um texto? desta forma:

    'Eu e o '+nomeMelhorAmigo+' vamos a uma festa amanhã.'

    Mas e se quisessemos colocar varias variaveis dentro de um texto...

    'Eu, '+nomeMelhorAmigo+', '+nomeMelhorAmigo1+', '+nomeMelhorAmigo2+', '+nomeMelhorAmigo3+', '+nomeMelhorAmigo4+' vamos a uma festa amanhã.'

    Com certeza você vai esquecer de abrir uma aspas ou colocar um sinal de +, isso seria um problema de execução da aplicação, então podemos usar Template Literals para nos ajudar, desta forma:

    `Eu, ${nomeMelhorAmigo}, ${nomeMelhorAmigo1}, ${nomeMelhorAmigo2}, ${nomeMelhorAmigo3}, ${nomeMelhorAmigo4} vamos a uma festa amanhã.`
