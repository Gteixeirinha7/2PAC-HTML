Vamos realmente aprender a fazer um for?

for, for invertido, forEach, mesclagem de listas Março 08, 2020

Aqui utilizaremos a linguagem C# para exemplificar

Como funciona a sintaxe do for:

for ({0}; {1}; {2}) {
    faz algo
}

  • {0} => Valor(es) a ser(em) instanciado(s), se houver mais de um, devem ser separados por vírgulas
  • {1} => Um comparador que resultará em um boolean, falso irá fazer com que o programa saia do loop, verdadeiro prosseguirá com o mesmo
  • {2} => Funciona como uma linha de código em um while, usado como incrementador, por exemplo: i++ fará com que i = 0 some 1 na variável, logo i = 1 na próxima iteração

    "For" e "while" são ambos loops e possuem funcionamentos iguais, mas são escritos de forma diferente:

    String[] lista = {"zero", "um", "dois"};

  • While

    • int i = 0;
    while (i < lista.Length) {
        Console.WriteLine(lista[i]);
         i++;
    }

  • For

    • for (int i=0; i < lista.Length; i++) {
        Console.WriteLine(lista[i]);
    }

    O exemplo acima é a forma mais básica para fazer um for, mas ainda não a mais simples

    Para simplificar, o foreach pode ser utilizado:

  • Foreach

    • foreach(String elemento in lista) {
        Console.WriteLine(elemento);
    }

    O segundo exemplo (foreach) funciona da mesma forma que o primeiro (for) e tem a vantagem de ser mais simples, porém um pouco mais limitado, pois não é possível saber o número da iteração atual.

    Em contrapartida não é possível obter o erro, list index out of bounds, muito conhecido por tentar acessar um elemento fora da lista.

    O próximo exemplo mostra um for invertido (do último para o primeiro):

    for (int i=lista.Length - 1; i>= 0; i--){
        Console.WriteLine(lista[i]);
    }

    Todos os trechos da sintaxe for são opcionais, então cuidado para não esquecer de colocar um incrementador ou condição de saída ou criará um loop infinito!

    O que NÃO deve ser feito:

    for (;;) { }
