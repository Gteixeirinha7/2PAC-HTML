Selecionando dados no banco orientado a objetos

Como funciona a sintaxe para selecionar, filtrar e ordenar registros em uma "Query" Abril 08, 2020

Aqui utilizaremos a linguagem Apex para exemplificar

Como funciona a sintaxe de uma Query no banco de dados do Salesforce:

SELECT {0} FROM {1} WHERE {2} ORDER BY {3}

  • {0} => Campos a serem selecionados, pode ser o código do registro, uma data de criação ou uma chave que dirá se o registro foi apagado.
  • {1} => A tabela, ou objeto no caso, que terá os registros selecionados.
  • {2} => Trecho de filtragem, podem ter diversos filtros, como por exemplo:

    DataCriacao < Data.hoje (selecionará todos os registros que não foram criados hoje).

  • {3} => Qual campo utilizar para ordenação dos registros, o padrão é do menor para o maior

    Exemplo: [SELECT Id, DataCriacao FROM Produto WHERE DataCriacao < Data.hoje ORDER BY DataCriacao]

    Considerando que 'hoje' fosse '2020/04/11' O retorno virá:

    [{Id='oo00fh1tf6ddrg186rht', DataCriacao=2020/04/02},

    {Id='oo00f645f6ddrgge6rht', DataCriacao=2020/04/05},

    {Id='o37161tf6ddrg186rht', DataCriacao=2020/04/10}]
